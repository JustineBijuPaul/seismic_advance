Project Overview

Seismic Quake is an advanced machine learning-based system designed to detect and analyze seismic activities through audio and waveform data. The project combines cutting-edge ML algorithms with real-time processing capabilities to provide accurate earthquake detection.

Key Features

1

Real-time Analysis

Process and analyze seismic data in real-time with 95% accuracy and 0.5s detection speed.

2

Multiple Format Support

Support for MSEED, WAV, and other standard seismic data formats.

3

Advanced Visualization

Interactive waveform visualization with highlighted seismic events.

ML Model Architecture

Feature Extraction

The system extracts MFCC features from audio signals using librosa:

• Mel-frequency cepstral coefficients (MFCCs)
• Spectral features
• Signal energy metrics

Model Training

Trained on a diverse dataset of seismic and non-seismic events using:

• Deep Neural Network
• Feature standardization
• Cross-validation
